# LV3 가장긴 팰린드롬

```
SOLUTION(타겟문자열):

    if 타겟문자열길이 == 1:
        return 1

    최대값 = -1e9

    # 길이 2 window - 길이 짝수

    for i 0~문자열길이-1
        왼쪽포인터, 오른쪽포인터 = i, i+1
        if 타겟문자열[왼쪽포인터] == 타겟문자열[오른쪽포인터]:
            while(1):
                if 타겟문자열[왼쪽포인터] != 타겟문자열[오른쪽포인터]:
                    break
                최대값 = 최대값계산(최대값, 오른쪽포인터-왼쪽포인터+1)
                오른쪽포인터 += 1
                왼쪽포인터 -= 1
                if 왼쪽포인터 < 0 or 오른쪽포인터 > 문자열길이-1:
                    루프탈출

    # 길이 3 window - 길이 홀수

    for i 0~문자열길이-2
        왼쪽포인터, 오른쪽포인터 = i, i+2
        if 타겟문자열[왼쪽포인터] == 타겟문자열[오른쪽포인터]:
            while(1):
                if 타겟문자열[왼쪽포인터] != 타겟문자열[오른쪽포인터]:
                    break
                최대값 = 최대값계산(최대값, 오른쪽포인터-왼쪽포인터+1)
                오른쪽포인터 += 1
                왼쪽포인터 -= 1
                if 왼쪽포인터 < 0 or 오른쪽포인터 > 문자열길이-1:
                    루프탈출


    if 최대값 == -1e9:
        return 1
    else:
        return 최대값






```
